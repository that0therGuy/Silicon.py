General:

1. Ever felt that online code editors always delay your work by constantly showing ads, popups or asking to login or signup.
If that is the case, this code editor is for you!
2. Run the code by pressing the top heading of the Editor.
3. Code is saved after it is run.

Editor:

4. Click inside the editor and press F10 to make editor go fullscreen.
5. numpy, scipy, pandas, matplotlib, sympy, scikit-learn, statsmodels, numba, matplotlib, bokeh, seaborn, biopython, rdkit, pyodide-py, lxml, pyyaml, dateutil, pytz, httpx, requests, websockets, scikit-learn, xgboost, micropip, pytest, setuptools and more support available baked right in the editor.
6. 'Scroll to' asks for a numerical input for scrolling to a specific line. If the operation is successful, it will blink green otherwise it will remain the same.
It executes automatically.
7. Press 'Ctrl + Left Mouse Click' to add a cursor for Multi-Cursor Editing.

8. Beta Features (if enabled):

1. Toggling false or true by clicking them in the editor
2. Swiping on number values to change them based on the swipe direction.
